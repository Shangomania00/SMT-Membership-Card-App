# SMT Loyalty - Digital Membership Card App

A fullstack digital loyalty platform for [Saif Mohammad Training Institute](https://www.smtuae.org).

## Features
- Member registration with digital stamp cards
- Unique barcode + QR code per member
- Admin dashboard with QR scanner for staff
- Push notifications to members
- OAuth 2.0 authentication

## Tech Stack
- React 19 + TypeScript + Tailwind CSS
- Three.js WebGL effects
- tRPC + Hono + Drizzle ORM + MySQL
- GSAP animations

## Pages
- `/` — Home (real SMT programs & workshops)
- `/signup` — Member registration
- `/card/:memberId` — Digital loyalty card
- `/admin` — Staff dashboard (QR scan, issue points, notifications)
